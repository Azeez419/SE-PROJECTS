# Code of Conduct – Campus Ride Project 

Our Pledge 

We are committed to creating a respectful, inclusive, and collaborative environment for everyone involved in the Campus Ride project. 
This code of conduct applies to all group members and contributors.


# Expected Behaviour 

All members are expected to: 

Treat others with respect and fairness 

Communicate  

Be open to feedback and constructive criticism 

Work collaboratively and support team members 

Take responsibility for assigned tasks 

Attend scheduled meetings and presentation days, or communicate in advance if unable to attend 



# Unacceptable Behaviour 

The following behaviour is not acceptable: 

Harassment, discrimination, or offensive language 

Personal attacks or disrespectful comments 

Deliberately blocking or delaying others’ work 

Plagiarism  

Misuse of project resources 

Repeated failure to attend meetings or presentation days without prior notice or valid reason 

Failure to communicate with the group regarding absences or missed responsibilities 

 

 



 

# Collaboration Guidelines 

Use GitHub responsibly (clear commits and meaningful commit messages) 

Review and test changes before pushing 

Discuss major changes with the group before merging 

Respect deadlines and group decisions 



# Reporting Issues 

Any issues or concerns should be raised respectfully within the group so they can be resolved fairly and promptly. 



# Enforcement and Consequences 

Failure to follow this code of conduct may result in the following actions, depending on the severity and frequency of the behaviour: 

First offence – Verbal or written warning and discussion with the group 

Second offence – Reassignment of tasks and formal documentation of the issue 

Repeated or serious offences (e.g. missing multiple meetings or presentation days without communication) – Removal from project contributions and escalation to the course instructor if required
